Webserv PHP CGI Examples

Welcome to the demo page for PHP CGI scripts running on our custom webserver implementation.

Available Examples:

  • Hello World
    A simple greeting script that demonstrates basic PHP functionality. Try adding a name parameter: /cgi-bin/hello.php?name=YourName
  • Server Information
    Displays CGI environment variables, HTTP headers, and request information. Useful for debugging.
  • Form Processing
    Demonstrates form handling with both GET and POST methods in PHP.

How It Works

These scripts showcase the integration between our webserver and PHP through the CGI interface. When you request a PHP script:

  1. The webserver recognizes the script as a CGI resource
  2. The server sets up environment variables and pipes for communication
  3. The PHP interpreter executes the script and returns the response
  4. The webserver forwards this response back to your browser

This implementation follows the CGI 1.1 specification, allowing seamless execution of PHP scripts on our custom server.
